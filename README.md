# D Co., 2/8 CAV — Archive Project
## Delta Company, 2nd Battalion, 8th Cavalry Regiment
## 1st Cavalry Division (Airmobile), Vietnam 1965–1972
## "Angry Skipper"

This repository is a primary source archive for Delta Company, 2/8 CAV.
It aggregates materials from veterans, families, and existing online resources
into a single organized, searchable, and permanently preserved collection.

**Initiated:** April 2026
**Compiled by:** Son of SGT Marvin Dale Miller, Cat Platoon, Dec 1970–Dec 1971

---

## Folder Structure

```
d28-cav-archive/
│
├── soldiers/                    # Individual soldier profiles and materials
```

## Status

All soldier profiles, events, and source documents in this repository
represent work in progress. Documents marked PLACEHOLDER are awaiting
additional source material. Documents marked PARTIAL have known gaps.

## Sensitive Information Notice

This repository is PRIVATE. The roster and contact research documents
contain personal contact information for living veterans. Do not make
this repository public without first removing or redacting all personal
contact details from the roster and research files.
