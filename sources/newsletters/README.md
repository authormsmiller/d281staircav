## How to Add Remaining Newsletters

1. Go to: https://www.angryskipperassociation.org/angry_skipper_association_newsletters.htm
2. Click each PDF link and download to your computer
3. Open the PDF and copy the text
4. Paste into the corresponding .md file in this folder
5. Add an Archive Notes section at the bottom with:
   - Names of all soldiers mentioned (with platoon and year if given)
   - Events cross-referenced with the main archive
   - Any new contact information found
   - Corrections to the roster or other records
